# Changelog

## v1.0.0 (2026-05-18)

首个开源正式版本。

### 功能

- **双编辑器** — Markdown（CodeMirror）和富文本（EditorJS）随时切换
- **Git 同步** — 支持 GitHub 和 Gitee，一键 push/pull，启动后自动拉取远端
- **版本历史** — 基于 Git，浏览提交记录，预览并恢复任意版本
- **全文搜索** — 跨笔记本即时搜索
- **内置终端** — 快速访问终端面板（Ctrl+`），自动定位到当前笔记目录
- **收藏与最近** — 置顶、星标、最近访问
- **多主题预览** — Markdown 预览支持多种主题
- **自动更新** — 基于 GitHub Releases 自动检测新版本
- **中英双语** — 界面语言可切换
