snowote 雪记

轻量、本地优先的笔记应用。支持 Markdown 与富文本编辑，Git 同步备份，版本历史随时恢复。

核心功能

📝

双编辑器

富文本编辑器（EditorJS）和 Markdown 编辑器（CodeMirror）随时切换，满足不同写作习惯。

🔄

Git 同步

支持 GitHub 和 Gitee，一键推送/拉取。启动时自动同步，数据永不丢失。

🕐

版本历史

基于 Git 的版本历史，浏览任意提交记录，随时预览并恢复历史版本。

🔍

全文搜索

跨所有笔记即时搜索，快速定位任何内容。

⌨

内置终端

按 Ctrl+` 唤出终端，自动定位到当前笔记所在目录，无需切换应用。

🌐

中英双语

界面支持中文和英文切换，适合不同语言习惯的用户。
