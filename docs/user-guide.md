# User Guide

## Quick Start

1. Download the latest release from [GitHub Releases](../../../releases)
2. Install and open snowote
3. Choose a folder as your notes directory
4. Start writing!

## Features

- **Markdown Editor** — Full CodeMirror editor with live preview
- **File Tree** — Browse and organize notes as files and folders
- **Git Sync** — Push/pull notes to GitHub or Gitee
- **Version History** — Browse and restore previous versions via Git
- **Full-text Search** — Search across all notes instantly
- **Favorites** — Pin and star important notes
- **Terminal** — Built-in terminal panel (Ctrl+`)
- **Scheduler** — Auto git-pull/push on a schedule

## Git Sync Setup

1. Open Settings → Sync tab
2. Choose GitHub or Gitee
3. Enter your username, repo name, and personal access token
4. Use the git sync button or configure auto-sync via Scheduler

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+\`` | Toggle terminal |
| `Alt+↑/↓` | Navigate between notes |
| `Ctrl+?` | Show keyboard shortcuts |
